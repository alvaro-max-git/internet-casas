## Cómo importar Seam en nuestro proyecto Maven

En pom.xml, dentro de `<dependencies>`

```xml
<dependency>
            <groupId>io.github.seamapi</groupId>
            <artifactId>java</artifactId>
            <!-- Sustituye 0.2.0 por la versión adecuada -->
            <version>0.2.0</version>
</dependency>
```
